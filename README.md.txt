# Projeto 3

Este é um projeto de uma lista de tarefas desenvolvido na linguagem de programação C, este projeto foi testado em um ambiente Windows no programa CodeBlocks.

## Replit

No ambiente Replit o programa foi testado, porém se tentarmos clicar no botão de "Run" um erro é disparado, pois há um problema interno com o compilador padrão do Replit, logo temos que rodar pelo "Shell" com o compilador `gcc`, primeiro vamos compilar o arquivo "main.c" com o comando:

`$ gcc main.c -o make.run`

Logo após devemos executar o arquivo compilado:

`$ ./make.run`

Depois da execução desses passos no terminal (Shell), o programa irá rodar normalmente.
